# Bridal Salon LUMIER

結婚相談所「Bridal Salon LUMIER」のホームページです。  
シンプルで上品なデザインを意識して制作しました。

## アプリURL
https://rina-11vp.onrender.com/

## 使用技術
- HTML
- CSS
- JavaScript
- Render

## 主なページ
- `/` トップページ
- `/about/` ルミエールのご紹介
- `/price/` 料金案内
- `/privacy/` 個人情報の取り扱いについて

## 工夫した点
- スマホ対応のレスポンシブデザイン
- 上品で見やすいUIデザイン
- シンプルな導線設計

## ファイル構成
- `assets/styles.css` : デザイン
- `assets/site.js` : メニュー動作
- `assets/images/` : 画像
